아크릴(Acrylic) 핸드백은 1950년대 루사이트(Lucite) 핸드백을 현대적으로 재해석한 작업이다. 이번 작업에서 레이저 가공의 단면 특성을 활용한 아크릴 상감 기법을 연구하였고 이를 패턴으로 활용하여 핸드백 디자인의 다양성을 추구하였다.

This acrylic handbag is a modern reinterpretation of the 1950s Lucite Handbag. The work is an outcome of studies on acrylic inlaying techniques which experiments with the cross section resulting from laser processing. Patterns created using this technique bring diversity in handbag design.
